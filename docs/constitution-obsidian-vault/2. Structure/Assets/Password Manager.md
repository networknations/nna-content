Shared credential management for Alliance accounts and platforms.

## Location
http://bitwarden.com/

## Description

The Password Manager securely stores and manages credentials for shared Alliance accounts — including social media, platforms, and services. It enables appropriate access while maintaining security.

## Custodian

[[Steward Council]]

## Access

- **View credentials:** Stewards (full access), designated Members (limited access as needed)
- **Add/modify:** Stewards
- **Administer:** Designated Steward(s)

## Security Practices

- Access granted on need-to-know basis
- Credentials rotated when Stewards depart
- Two-factor authentication enabled where possible
- Access logs maintained
