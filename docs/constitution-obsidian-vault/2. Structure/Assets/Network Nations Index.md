A living directory of projects, communities, organizations, and resources related to Network Nations.

## Location
https://github.com/network-nations/website

## Description

The Index serves as the central discovery mechanism for the movement — helping people find Network Nations, related projects, learning groups, hubs, research, educational materials, and governance toolkits. It operates as a public repository with contributions via GitHub pull requests and light moderation for alignment with the [[Network Nation Definition]].

## Custodian

[[Steward Council]] maintains the Index infrastructure and coordinates moderation.

## Access

- **View:** Public
- **Contribute:** Open to anyone via pull request
- **Moderate:** Stewards per [[Index Moderation Protocol]]

- Learning groups and hubs
- Research and case studies
- Educational resources and guides
