The governing document of the Network Nations Alliance.

## Location
https://www.networknations.network/agreement

## Description

This Constitution defines the Alliance's identity, structure, and protocols. It is the foundational agreement that all members affirm upon joining.

## Custodian

[[Steward Council]] maintains the Constitution document and coordinates amendment processes.

## Access

- **Read:** Public
- **Propose amendments:** Any Member
- **Approve amendments:** Alliance Assembly per [[3. Protocols/Asset Protocols/Constitution Amendment Protocol|Constitution Amendment Protocol]]
